# LUMI DBS

- An easy tool for managing data on game development.

## ROADMAP
- Generate Schemas
- Populate data based on the Schemas
- JSON Viewer and Editor
- Export
  - Select Tables to export
  - Split or Mono File option
  - JSON Format
  - GDScript Format ( Godot Game Engine)
  - CSV
- Import
  - JSON
  - GDScript
  - CSV
- Project File
  - Open/Save
- Config File
- History Log
  - Undo button
- Styling
